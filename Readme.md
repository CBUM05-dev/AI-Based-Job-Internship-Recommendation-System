# Job Recommendation AI System

## Description
Context-aware hybrid AI system for recommending jobs and internships using ML + LLMs.

## Features
- User text understanding via LLM
- Feature engineering & ML ranking
- Top-K job recommendations
- Explanations via LLM

## Installation
1. `git clone <repo>`
2. `python -m venv venv && source venv/bin/activate`
3. `pip install -r requirements.txt`

## Usage
- Run API: `uvicorn api.main:app --reload`
- Call `/recommend` endpoint with user input
